## The Challenge

Read analog rotation, convert it to digital data, and present it to a computer as a standard game controller.

## The Build

I wrote C++ in the Arduino IDE using the Unojoy framework to map a potentiometer's rotation and custom buttons to controller axes.

## Skills Demonstrated

Microcontrollers, analog-to-digital I/O, and device interfacing.
